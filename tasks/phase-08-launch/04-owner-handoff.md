# Task: Owner Handoff & Training

**Phase:** 8 — QA & Launch  
**Week:** 12

## Goal

Train the business owner and staff to independently manage products, orders, and basic site content.

## Requirements

- Walkthrough of admin dashboard:
  - Adding/editing products, variants, and images
  - Managing categories
  - Processing orders (status updates)
  - Understanding low-stock alerts
- Document common admin tasks in concise guide
- Verify owner can:
  - Add a new product with variants and images
  - Update an order status
  - View dashboard metrics
- Provide support contact for post-launch issues

## Acceptance Criteria

- [ ] Owner successfully adds a product independently
- [ ] Owner successfully updates an order status
- [ ] Owner understands draft vs active products
- [ ] Staff (if applicable) can access allowed admin features
- [ ] Admin task guide written and shared with owner
- [ ] Post-launch support plan communicated

## Dependencies

- [03-production-deployment.md](./03-production-deployment.md)
- All admin features complete (Phases 2, 6)

## Files to Modify

| File | Action |
|---|---|
| `docs/admin-guide.md` | Create |
| `docs/deployment.md` | Update (post-launch section) |

## Definition of Done

- [ ] Training session completed with owner
- [ ] Admin guide document created
- [ ] Owner confirms confidence in daily operations
- [ ] Project marked as launched
